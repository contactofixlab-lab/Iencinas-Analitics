# PropDash Analytics 📊

Sistema integral de dashboards para inmobiliaria con control de acceso por roles.

## 🚀 Características

✅ Sistema de autenticación con 4 usuarios predefinidos
✅ Control de acceso por roles y departamentos
✅ 4 módulos de dashboards (Finanzas, Comercial, Marketing, Valor Empresa)
✅ Interfaz moderna y responsiva
✅ Navbar con perfil de usuario
✅ Sidebar con navegación intuitiva

## 🛠️ Instalación Local

### Prerequisites
- Node.js 16+ 
- npm o yarn

### Pasos

1. **Clonar el repositorio**
```bash
git clone <tu-repo-url>
cd propdash-analytics
```

2. **Instalar dependencias**
```bash
npm install
```

3. **Ejecutar en desarrollo**
```bash
npm run dev
```

4. **Abrir en el navegador**
```
http://localhost:3000
```

## 🌐 Desplegar a Vercel

### Opción 1: Con GitHub (Recomendado)

1. Sube el código a GitHub
2. Ve a [vercel.com](https://vercel.com)
3. Haz clic en "New Project"
4. Selecciona tu repositorio
5. Haz clic en "Deploy"
6. ¡Listo!

### Opción 2: Con Vercel CLI

```bash
npm i -g vercel
vercel
```
